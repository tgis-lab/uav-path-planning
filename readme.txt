1.首先使用voxel_grid.py进行三维栅格转化将glb转化成ply文件并进行可视乎（其实主要目的是获得可视化图像）
A_star_glb_pointcloud.py是利用A*算法进行点对点路径规划，可以得到点对点间的规划路径以及一个grid.npy文件（这个npy文件就是三维栅格地图用0 1表达的文件，0代表未占据，1代表已占据）
2.得到grid.npy后使用2Dsplit.py得到固定高度的二维栅格地图
3.使用BCD.py文件对获得的二维栅格地图进行子区域划分
4.使用MAS+LOS.py进行子区域覆盖路径规划
5.使用SG.py获得每个子区域的无人机起始点和终点
6.使用Astar.py获得不同子区域起点到终点规划的路径、路径长度，得到一个距离矩阵
7.使用Distence.py获得每个子区域覆盖路径长度并添加到距离矩阵中
8.使用SA+ACO进行任务分配
